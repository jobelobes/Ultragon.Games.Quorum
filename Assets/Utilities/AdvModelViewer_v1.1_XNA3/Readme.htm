Adventures XNA Model Viewer Readme

Adventures Model Viewer is a simple 3D model and effect viewing tool.
This tool is for content artists to test models and textures without using Visual Studio.

Features 1.0

  • Load and build FBX and X model files
  • Load compiled XNB content files
  • Render using BasicEffect or custom effect files
  • Render solid models or wireframe
  • Rotate model and zoom
  • Load HLSL effect files
  • Edit effect parameters
  • Supports effect semantics, such as WORLD and COLOR (TODO: add full semantic list)
  • View custom effect code
  • Load textures from raw or compiled files
  • Drag and drop models and effects for quick loading

Requirements

  • Microsoft XNA Game Studio 3.0 (Because the content pipeline is not included in the runtime redistributables)
  • Microsoft .NET 3.5 Framework

Future Features (not supported in this version)

  • Load 3D and Cubemap textures
  • Unload textures applied to custom effects
  • Helper UI for controlling lights
  • Helper UI for editing vector parameters etc. (other than semantic based parameters)
  • View model information, vertex and poly count etc.
  • Create help text and maybe a small "tutorial"
  • Drag and drop for quick loading textures

Random ideas of improvements
